# ADR-001: Stack tecnológico inicial (Cliente-Servidor)

## Estado
Propuesto

## Contexto
Se requiere implementar la arquitectura Cliente-Servidor para el proyecto SalUD.

## Decisión
- Frontend: React
- Backend: Node.js
- Base de datos: Por definir
- Contenedores: Docker
- Despliegue: Azure

## Consecuencias
- Permite separar claramente cliente y servidor.
- Facilita escalabilidad y despliegue en la nube.
- El equipo debe acordar contrato API y esquema de roles/permisos.
